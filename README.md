# XFlow

Multi-agent AI system with x402 payment routing on X Layer.

## Architecture
```
User / External Agent
    ↓ x402 (any chain)
x402 Payment Adapter
    ↓
Orchestrator
    ↓
CEX Agent / DEX Agent (OKX OnchainOS)
    ↓
X Layer onchain
```

## Features
- x402 multi-chain payment adapter
- AI orchestrator (CEX/DEX routing)
- OKX OnchainOS Skills integration
- X Layer onchain activity recording
- Agent Economy Dashboard

## Built with
- x402 Protocol
- OKX OnchainOS
- X Layer (eip155:196)
- payai facilitator
